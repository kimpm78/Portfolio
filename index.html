MY_STORY

  2012年からデザインに興味を持ち、自らデザインし、製作することを楽しみました。様々な製品デザインの経験を通じて、ウェブデザインにも興味を持ち、ウェブデザインの基礎を学び、コーディングを始めました。
徐々にプログラミングを通じて作品を生み出すことから大きな達成感を感じ、それによってプログラミングの世界へ一歩踏み入ることができました。

  • ・・・

[ 2015 - 2021 作品 ]

  インターンから始まり、さまざまな分野での参加を経験した作品です。 このようなデザイン経験は、プログラマーとしての成長に役立ち、言語やアルゴリズムを学ぶことで、デザイナーが望む方向を理解するようになりました。
 常に最新のトレンドに沿って新しい技術を学ぶことができるということに達成感を感じます。常に不足している部分を補い、プログラムに関連したことを完璧にするために、絶えず勉強しています。

TECH STACK

アイコンにマウスを合わせると、詳細な説明が表示されます。

# FRONT_END SKILL

  • HTML5

    WEBアクセシビリティと標準準拠を遵守し、セマンティックタグを使用してマークアップすることができます。

  • CSS3

    positionおよびdisplayプロパティを適切に活用して、レスポンシブWebを適用することができます。

  • SASS

    SCSSの文法とモジュール化をうまく活用して、スタイル作業をスムーズに行うことができます。

  • Javascript

    JavaScriptの配列、条件分岐、メソッド、DOMを理解し、さまざまな状況に適用することができます。

  • Typescript

    Javascriptよりコードの可読性を高め、デバッグを容易にするために適用しています。

  • React.js

    構造と各種なフックについてよく知っており、汎用性が高く再利用可能なコンポーネントの設計ができます。

# BACK_END SKILL

  • Node.js

    サーバーのAPIを構築し、DBと連携してCRUD機能を実行することができます。

  • Next.js

    SSRと静的サイト生成を利用して、効率的にWEBサイトを構築や高度な機能などを開発生産性と最適化に努めます。

  • mongoDB

    mongooseを使用してMongoDBのスキーマを定義し、データを処理することができます。

  • MySQL

    データベースの作成、変更、クエリの実行、ユーザー権限の管理など、各種な機能を設定することができます。

  • JSON

    JSONのオブジェクト構造を理解し、JavaScriptで活用することができます。

# USING_TOOL

  • VS Code

    主に使用しているコーディングツールで、コーディングを便利にする拡張機能をインストールして利用しています。

  • Figma

    デスクトップやモバイルのUI/UXデザインを作成し、デバッグをうまく活用することができます。

  • PhotoShop

    マスク効果やレイヤーの活用などを使って、さまざまなグラフィック要素を制作することができます。

  • Illustrator

    ペンツールを活用し、その他のツールを使って、ベクター形式のデザイン要素を制作することができます。

  • POSTMAN

    ペンツールを活用し、その他のツールを使って、ベクター形式のデザイン要素を制作することができます。

  • Git

    commit、push、pull、cloneなどのGitコマンドを使用して、リポジトリにアクセスできます。
